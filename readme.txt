Instrucciones para la ejecucion del programa:
1. Abra sql developer y ejecute el archivo "EsquemaAlohAndes" que se encuentra en la carpeta data de este proyecto
2. Abra el proyecto en Eclipse 
3. Ejecute el main del proyecto "InterfazAlohAndesApp"
4. Espere a que cargue la aplicacion
5. Navegue y use la funcionalidad
6. Al terminar haga click en el menu "Mantenimiento" y seleccione la opcion "cerra base de datos" antes de cerrar la aplicacion


En caso de ser necesario limpiar la base de datos ejecute en sql developer el archivo "Limpiar AlohAndes"
